Card

Card component used to group information category. By default card will have 24px padding sides ($spacing-5) and 16px padding ends ($spacing-4)

Card contains 3 type of variations

  • Bordered card
  • Bordered card dark
  • Elevated card 1dp
  • Elevated card 2dp

By default when declared .card class Elevated card 1dp will be called. Add extra classname .card-border for Bordered card, .card-border-dark for Dark border card and .card-2dp for Elevated card 2dp

The usage of this card is vary depends on design scenarios. As mention, the default card will be most commonly used for wrapping content item, and bordered style will use within the default card

Avatar

Avatar used to represent the profile for Brio users. Avatar can contains different sizes XXS, XS, SM, MD, LG. By default, if calling .avatar the XS size will be used.

If there is no profile image of the user, the initial will be displayed by calling the avatar attribute of data-content from parent avatar.

To show that avatar clickable add attribute clickable so that the cursor change to pointer. See example below

Banner

Banner is used to hightlight an important message within a page, within a card component or modal.

There are 5 variant of banner

  • Green - to indicates success message or positive item
  • Blue - to indicates informative message or hint
  • Red - to indicates error message, alert or danger
  • Orange - to indicates warning message or warning
  • Purple - to indicates guide message or documentation

Banner can contain standalone text or together with icon to strongly support the corresponding banner descriptions. It has to be displayed on top of white background component (card, modal) and need to be a block element.

Button Group

Button group can be used to provide fast selection of choices. The choices must be within 2 - 4 items. If the choices contain more than 4 it is not going to be good usage to use the button group.

To retain the active selection, the classname .is-active need to bind during the button choices clicked

There are 2 types of button group

  • Normal button group - Common usage for button group
  • Custom button group - Different active color based on button type assigned (CTA,Primary, Danger)
